# 📊 Contador de Dias de Estudo

Projeto simples em JavaScript para registrar e acompanhar dias de estudo.

## 🚀 Funcionalidades

- Adicionar um novo dia de estudo
- Evitar registro duplicado no mesmo dia
- Listagem dos dias estudados com data
- Persistência de dados com localStorage
- Feedback ao usuário com mensagens dinâmicas
- Limpeza automática do feedback após alguns segundos

## 🧠 Conceitos aplicados

- Manipulação de DOM
- Gerenciamento de estado
- Separação de responsabilidades (lógica vs renderização)
- Uso de arrays e métodos como `.some()`
- Persistência com `localStorage`
- Uso de `setTimeout` para controle de tempo
- Atualização reativa da interface baseada no estado

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript

## 💡 Observações

A interface é controlada pelo estado da aplicação, garantindo consistência entre dados e visualização.
